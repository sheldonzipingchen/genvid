# TikTok UGC 视频生成平台 - 市场调研报告

## 📊 市场概述

### 目标用户画像
- **跨境电商卖家**：需要快速制作 TikTok/Reels/Shorts 广告视频
- **独立站品牌**：需要大量 UGC 风格内容进行 A/B 测试
- **营销代理商**：需要为客户提供高频内容产出
- **小型电商团队**：预算有限，无法承担传统 UGC 制作成本

### 市场痛点
1. **成本高**：传统 UGC 创作者收费 $50-800/视频
2. **周期长**：从brief到交付需 1-3 周
3. **不可控**：创作者档期、质量问题、返修困难
4. **规模化难**：难以快速产出 20-50 个变体进行测试
5. **语言障碍**：跨境卖家难以制作多语言内容

## 🗄️ 数据结构推断 (Schema Inference)

### 数据库表设计 (PostgreSQL)

```sql
-- 用户表
CREATE TABLE profiles (
  id UUID PRIMARY KEY REFERENCES auth.users(id),
  email VARCHAR(255) NOT NULL,
  full_name VARCHAR(100),
  avatar_url TEXT,
  credits_remaining INTEGER DEFAULT 3,
  subscription_tier VARCHAR(20) DEFAULT 'free', -- free, starter, pro, enterprise
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- 视频项目表
CREATE TABLE projects (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES profiles(id) NOT NULL,
  title VARCHAR(200),
  status VARCHAR(20) DEFAULT 'draft', -- draft, processing, completed, failed
  product_name VARCHAR(200),
  product_description TEXT,
  product_url TEXT,
  script TEXT,
  avatar_id UUID REFERENCES avatars(id),
  video_url TEXT,
  thumbnail_url TEXT,
  duration_seconds INTEGER,
  format VARCHAR(10) DEFAULT '9:16', -- 9:16, 1:1, 16:9
  language VARCHAR(10) DEFAULT 'en',
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW(),
  completed_at TIMESTAMPTZ
);

-- 素材表
CREATE TABLE assets (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  project_id UUID REFERENCES projects(id) ON DELETE CASCADE,
  type VARCHAR(20) NOT NULL, -- image, video, audio
  url TEXT NOT NULL,
  filename VARCHAR(255),
  file_size INTEGER,
  mime_type VARCHAR(100),
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Avatar 库表
CREATE TABLE avatars (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name VARCHAR(100) NOT NULL,
  gender VARCHAR(10), -- male, female, other
  age_range VARCHAR(20), -- 20s, 30s, 40s, 50s+
  ethnicity VARCHAR(50),
  language VARCHAR(10)[] DEFAULT ARRAY['en'],
  style VARCHAR(50), -- casual, professional, energetic
  preview_video_url TEXT,
  thumbnail_url TEXT,
  is_premium BOOLEAN DEFAULT false,
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 脚本模板表
CREATE TABLE script_templates (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name VARCHAR(100) NOT NULL,
  category VARCHAR(50), -- review, unboxing, tutorial, comparison
  template_text TEXT NOT NULL,
  language VARCHAR(10) DEFAULT 'en',
  is_premium BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 订阅/交易表
CREATE TABLE subscriptions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES profiles(id) NOT NULL,
  plan_id VARCHAR(50) NOT NULL, -- starter_monthly, pro_monthly, etc.
  status VARCHAR(20) DEFAULT 'active', -- active, canceled, expired
  credits_per_month INTEGER,
  credits_used_this_month INTEGER DEFAULT 0,
  current_period_start TIMESTAMPTZ,
  current_period_end TIMESTAMPTZ,
  stripe_subscription_id VARCHAR(100),
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

### RLS (Row Level Security) 策略

```sql
-- 用户只能访问自己的数据
ALTER TABLE profiles ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Users can view own profile" ON profiles
  FOR SELECT USING (auth.uid() = id);
CREATE POLICY "Users can update own profile" ON profiles
  FOR UPDATE USING (auth.uid() = id);

-- 用户只能访问自己的项目
ALTER TABLE projects ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Users can manage own projects" ON projects
  FOR ALL USING (auth.uid() = user_id);

-- Avatar 对所有认证用户可见
ALTER TABLE avatars ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Avatars are viewable by all authenticated users" ON avatars
  FOR SELECT USING (auth.role() = 'authenticated' AND is_active = true);
```

### 视频生成技术选项

#### 方案 A: HeyGen API 集成
- **优点**: 质量最高，开箱即用
- **缺点**: 成本高($0.1-1/秒)，依赖第三方
- **适合**: MVP 阶段快速验证

#### 方案 B: 自建 AI 视频生成 (基于开源模型)
- **技术**: SadTalker + Wav2Lip + TTS
- **优点**: 成本可控，可定制
- **缺点**: 开发周期长，需要 GPU 资源
- **适合**: 长期规划

#### 方案 C: 混合方案
- MVP 使用 HeyGen/Synthesia API
- 同步开发自建能力
- 逐步迁移降低成本
